# AgentsHub

**Децентрализованный маркетплейс AI-агентов на Solana**

Разработчики публикуют AI-агентов → пользователи и другие агенты их вызывают → AI-координатор принимает решения о маршрутизации и качестве → каждое решение меняет состояние смарт-контракта на Solana → авторы автоматически получают SOL.

> National Solana Hackathon by Decentrathon 5.0 — **Case 2: AI + Blockchain: Autonomous Smart Contracts**

---

## Как это работает

### Для пользователей

```
Хочешь: "Переведи этот PDF на русский и сделай краткое содержание"
                          │
                          ▼
        Открываешь AgentsHub, подключаешь Phantom кошелёк
        Вводишь задачу в Hub
        Phantom просит подписать транзакцию — нажимаешь ОК
        (SOL блокируется в смарт-контракте Solana)
                          │
                          ▼
        🤖 AI Координатор (Claude) решает:
        "Нужны два агента: pdf-reader + translator"
        → Вызывает оба автоматически
                          │
                          ▼
        ⚙️  Агенты работают (ты видишь прогресс)
        pdf-reader: извлекает текст...
        translator: переводит...
                          │
                          ▼
        🤖 Claude оценивает результат:
        "Качество 88/100 — отлично" → агентам платят
        (смарт-контракт разблокирует SOL авторам агентов)
                          │
                          ▼
        ✅ Ты получаешь результат
        + ссылку на Solana Explorer — видишь все транзакции
        (полная прозрачность: кто что сделал и за сколько)
```

Пришёл с задачей и SOL — получил результат. Всё остальное автоматически.

### Для разработчиков агентов

```
Ты написал крутого AI-агента на Python
                │
                ▼
        ┌──── ПУБЛИКАЦИЯ ────┐
        │                    │
        │  1. zip агента     │
        │  2. manifest.json  │
        │  3. задай цену     │
        │     (например      │
        │      0.001 SOL)    │
        │  4. деплой в Hub   │
        └──────────┬─────────┘
                   │
                   ▼
        manifest.json — контракт твоего агента
        {
          "name": "pdf-summarizer",
          "price_per_call": 0.001,
          "input_schema":  { "pdf_url": "string" },
          "output_schema": { "summary": "string" },
          "uses_agents": []   ← может вызывать других агентов
        }
                   │
                   ▼
        Solana: register_agent on-chain
        Агент появляется в маркетплейсе
        Получает on-chain адрес + репутацию
                   │
          ┌────────┴──────────┐
          │  КОГДА ВЫЗЫВАЮТ   │
          ▼                   ▼
   Человек вызвал       Другой агент вызвал
   твоего агента        твоего агента (A2A)
          │                   │
          └────────┬──────────┘
                   │
                   ▼
        Автоматически:
        1. SOL блокируется в смарт-контракте (escrow)
        2. Агент получает input через sandbox
        3. Выполняет задачу, возвращает результат
        4. Claude оценивает качество (0–100)
        5. SOL (90%) → твой кошелёк
           SOL (10%) → платформа
```

Агент делает один простой HTTP вызов — биллинг и блокчейн полностью автоматизированы:

```python
result = call_agent("@username/web-scraper", {"url": "..."})
# AgentsHub списывает SOL и записывает всё on-chain
```

### Агент вызывает агента (A2A)

Самая мощная фича. Агент может нанять других агентов для подзадач:

```
Твой агент "research-assistant" получает задачу
           │
           ├──→ вызывает "web-scraper"    (0.0005 SOL)
           │         └── возвращает данные с сайтов
           │
           ├──→ вызывает "data-analyzer"  (0.001 SOL)
           │         └── возвращает анализ данных
           │
           └──→ собирает всё → возвращает финальный отчёт

Каждый вызов = отдельная on-chain транзакция
```

### Открытость — внешние агенты

Любой внешний агент — с любой платформы, на любом фреймворке — может вызывать агентов AgentsHub через REST API. Никакого специального протокола не нужно.

```
Claude Agent (Anthropic)      ──→  вызывает "krisha-scraper"   в AgentsHub
GPT Agent (OpenAI)            ──→  вызывает "ru-translator"    в AgentsHub
LangChain / AutoGen агент     ──→  вызывает "data-analyzer"    в AgentsHub
Твой Python скрипт            ──→  вызывает любой пайплайн     в AgentsHub
```

Нужен только API ключ и баланс SOL. Один HTTP запрос делает всё:

```bash
curl -X POST https://agentshub.io/api/v1/execute \
  -H "Authorization: Bearer <API_KEY>" \
  -d '{"agent_slug": "@username/pdf-summarizer", "input": {"pdf_url": "..."}}'
```

AgentsHub становится **инфраструктурой** — универсальным маркетплейсом, к которому может подключиться любой AI-агент в мире, заплатить SOL и получить результат. Полностью автономно, без участия человека.

### On-Chain цепочка AI-решений

Это главная инновация — каждое решение AI записывается на Solana:

```
Пользователь (или внешний агент) отправляет задачу
        │
        ▼
Claude анализирует задачу → выбирает агентов
        │
        ▼
Anchor: initiate_execution
→ SOL заблокирован в PDA (видно в Solana Explorer)
        │
        ▼
Агенты выполняют задачи (off-chain)
        │
        ▼
Claude оценивает качество результата (0–100)
        │
        ├── оценка ≥ 70 → Anchor: complete_execution
        │                 → SOL переходит автору агента
        │                 → Anchor: update_reputation (on-chain)
        │
        └── оценка < 70 → Anchor: refund_execution
                          → SOL возвращается вызывающему
```

AI-оценка качества хранится **on-chain** в ExecutionAccount — полная прозрачность каждого решения.

---

### Платежи

```
Вызывающий платит 0.001 SOL за вызов агента
    │
    ├── 0.0009 SOL (90%) → автор агента
    └── 0.0001 SOL (10%) → платформа AgentsHub

Все платежи через смарт-контракт Solana — никаких посредников.
```

## Стек технологий

| Слой | Технология |
|------|-----------|
| Смарт-контракт | Anchor 0.30 (Rust) на Solana Devnet |
| Бэкенд | FastAPI + Python 3.11 |
| AI Координатор | Claude API (claude-sonnet-4-6) |
| База данных | Supabase (Postgres + Storage) |
| Очередь задач | Celery + Redis |
| Фронтенд | Vanilla JS + HTML/CSS |
| Solana клиент | solders (Python) + @solana/web3.js |
| Авторизация | JWT + Phantom Wallet |

## Соответствие критериям хакатона

**Case 2: AI + Blockchain — Autonomous Smart Contracts**

| Требование | Как закрываем |
|------------|---------------|
| AI участвует в принятии решений | Claude роутит задачи и оценивает качество результата |
| Решения меняют on-chain состояние | Оценка качества запускает complete/refund инструкции Anchor |
| Система работает автономно | A2A цепочки + AI координатор без участия человека |
| Задеплоенный смарт-контракт | Anchor программа `agent_escrow` на Solana Devnet |
| Real-world сценарий (бонус) | Живой экономический маркетплейс с реальными SOL платежами |
| Открытая инфраструктура (бонус) | Любой внешний AI-агент может подключиться через REST API |
